## EasyDone

## 📌 Sobre o Projeto
O EasyDone é uma solução tecnológica que visa ajudar as pessoas a se organizarem melhor.O sistema oferece uma ferramenta funcional e centralizada que permite ao usuário inserir, visualizar e controlar o status de suas tarefas cotidianas de forma clara e objetiva. 

Este projeto foi desenvolvido como um trabalho apresentado à disciplina de Fábrica de Software como requisito para avaliação, no curso de Engenharia de Software da Univille.

## 🎯 Público-Alvo
O sistema é destinado a estudantes, profissionais de diversas áreas (como Tecnologia da Informação e administração), e pessoas com rotinas intensas que precisam de uma ferramenta simples para gerenciar suas atividades.

## ✨ Funcionalidades
* Cadastrar novas tarefas por meio de um campo de texto.
* Listar todas as tarefas adicionadas em uma interface visual.
* Marcar tarefas individualmente como concluídas.

## 🛠️ Tecnologias Utilizadas
* **HTML:** Para a estruturação dos elementos da página.
* **CSS:** Para a estilização e layout da interface.
* **JavaScript:** Para a lógica de manipulação da lista e interação do usuário.

## 🚀 Como Executar
O sistema opera de forma totalmente isolada e possui acesso livre e imediato, sem a necessidade de login. Os dados são mantidos apenas na memória da sessão atual.

Para rodar o projeto localmente:
1. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na mesma pasta.
2. Dê um duplo clique no arquivo `index.html` para abri-lo em seu navegador padrão.
3. Não é necessária nenhuma instalação de dependências ou configuração de servidor.
