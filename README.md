## EasyDone

## 📌 Sobre o Projeto
[cite_start]O EasyDone é uma solução tecnológica que visa ajudar as pessoas a se organizarem melhor[cite: 12]. [cite_start]O sistema oferece uma ferramenta funcional e centralizada que permite ao usuário inserir, visualizar e controlar o status de suas tarefas cotidianas de forma clara e objetiva[cite: 25]. 

[cite_start]Este projeto foi desenvolvido como um trabalho apresentado à disciplina de Fábrica de Software como requisito para avaliação [cite: 5][cite_start], no curso de Engenharia de Software [cite: 2] [cite_start]da Univille[cite: 1].

## 🎯 Público-Alvo
[cite_start]O sistema é destinado a estudantes, profissionais de diversas áreas (como Tecnologia da Informação e administração), e pessoas com rotinas intensas que precisam de uma ferramenta simples para gerenciar suas atividades[cite: 22].

## ✨ Funcionalidades
* [cite_start]Cadastrar novas tarefas por meio de um campo de texto[cite: 29].
* [cite_start]Listar todas as tarefas adicionadas em uma interface visual[cite: 29].
* [cite_start]Marcar tarefas individualmente como concluídas[cite: 30].

## 🛠️ Tecnologias Utilizadas
* [cite_start]**HTML:** Para a estruturação dos elementos da página[cite: 52].
* [cite_start]**CSS:** Para a estilização e layout da interface[cite: 53].
* [cite_start]**JavaScript:** Para a lógica de manipulação da lista e interação do usuário[cite: 54].

## 🚀 Como Executar
[cite_start]O sistema opera de forma totalmente isolada e possui acesso livre e imediato, sem a necessidade de login[cite: 47, 49]. [cite_start]Os dados são mantidos apenas na memória da sessão atual[cite: 47].

Para rodar o projeto localmente:
1. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na mesma pasta.
2. Dê um duplo clique no arquivo `index.html` para abri-lo em seu navegador padrão.
3. Não é necessária nenhuma instalação de dependências ou configuração de servidor.
